# Couch to 5K

A clean, browser-based Couch to 5K training app. No install needed — open the HTML file in any browser and start running.

## Features

- **Full 9-week plan** with the authentic C25K interval structure
- **Live workout timer** with a circular progress ring and run/walk badge
- **Week-by-week progress grid** — runs unlock as you complete each one
- **Dot timeline** showing your position through all intervals during a session
- **Confetti celebration** on run completion
- **Light and dark mode** with system preference detection and manual toggle
- **Mobile-first** design, works great on any screen size

## How to use

1. Open `couch-to-5k.html` in your browser
2. Tap the green card to start your next run
3. Follow the run/walk prompts during your workout
4. Mark the run complete when done

## Tech

Single-file HTML/CSS/JS — no frameworks, no build step, no dependencies except Google Fonts.

Built with:
- DM Serif Display + DM Sans (Google Fonts)
- Vanilla JS (ES2020)
- CSS custom properties + OKLCH colour space
- SVG inline logo
